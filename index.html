Sobre mim

Meu nome é Vitor Augusto de Mattos e eu sou um apaixonado por tecnologia que decidiu trilhar o caminho da programação, tenho 28 anos, desde pequeno tive paixão por tecnologia. Quando enfim comecei a estudar sobre, e estou seguindo neste ramo.

Sempre fui uma pessoa curiosa e que gosta de solucionar problemas, e aprender essas e outras características me aproximaram cada vez mais ao amor pela tecnologia. Atualmente estou cursando Analíse e Desenvolvimento de Sistemas .

Atualmente estou focando nas tecnologias voltadas ao FrontEnd, gosto muito de trabalhar com javascript | HTML | CSS, estou aprendendo sobre TypeScrip, React, Node.JS, me familiarizando com base de dados MySQL/MariaDB. Estas linguagens para não depender de um Dev Front nem de um Dev Back.

Habilidades

Minhas habilidades criativas e experiencias

HTML

É a linguagem de marcação padrão do World Wide Web. Utilizada na construção de páginas web.

CSS

É utilizada para especificar a forma como um documento HTML deve ser apresentado em um navegador.

JavaScript

Javascript é uma linguagem de programação interpretada, ou seja, ela não precisa de um arquivo para ser executada.

Node.JS

Node.js é um software de código aberto, multiplataforma que permite a execução de códigos JavaScript fora de um navegador web.

React

React é uma biblioteca JavaScript de código aberto para criar interfaces de usuário.

TypeScript

É um superconjunto sintático estrito de JavaScript e adiciona tipagem estática opcional à linguagem

MySQL

É um software de banco de dados relacional de código aberto, desenvolvido pela MySql AB.

MariaDB

MariaDB é um sistema de gerenciamento de banco de dados que surgiu como fork do MySQL
